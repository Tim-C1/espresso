# Experienced Retrieval Hard

This five-page adversarial benchmark contains 40 body sentences. Every page
mixes established background, one known-but-important-sounding freshness claim,
a bridge from standard operations to selective refresh, two true quantitative
deltas, and two low-value operational/commercial sentences.

The near misses intentionally mention an interest phrase while stating that the
configuration was already validated. Thresholds allow some classifier error but
bound false delta promotion, familiar suppression, recall, and guidance density.
