# Experienced Retrieval Engineer

This synthetic two-page fixture tests the product promise for a reader who
already understands vector search and retrieval-augmented generation.

The document deliberately mixes:

- familiar background about embeddings, nearest-neighbor search, and RAG;
- bridge claims connecting known indexing and monitoring workflows to new ideas;
- two quantitative delta claims about freshness and reindexing;
- low-value deployment and vendor material that should remain unobtrusive.

`baseline.json` describes the reader without encoding sentence IDs or expected
labels. `gold_labels.json` is the independent sentence-level scoring rubric.
The PDF is synthetic public test data and contains no private or provider output.
